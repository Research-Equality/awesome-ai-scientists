# Roadmap

This repository is being built as a long-lived map, not a one-shot list.

## Phase 1: Foundation

- [x] Define the layered taxonomy.
- [x] Define the compact tag ribbon and extended tag system.
- [x] Establish README information architecture.
- [x] Add contribution and inclusion rules.
- [x] Add baseline CI for linting and link checking.

## Phase 2: Coverage expansion

- [ ] Expand domain-specific sections in biology, chemistry, materials, medicine, physics, and math.
- [ ] Add more research-platform and orchestration coverage.
- [ ] Deepen workflow-module coverage beyond literature, experiment, and writing.
- [ ] Add a lightweight archived section for historically important but inactive resources.

## Phase 3: Stronger curation signals

- [ ] Track `Maturity` more systematically.
- [ ] Add short comparison notes for crowded clusters.
- [ ] Document deprecation and replacement policies.
- [ ] Introduce a lightweight change log for major taxonomy updates.

## Phase 4: Machine-readable layer

- [ ] Add a structured data file for entries.
- [ ] Generate tag-based views automatically.
- [ ] Enable easier ecosystem analysis and downstream reuse.

## Contribution ideas

Useful next steps from contributors:

- Suggest missing canonical systems.
- Improve category placement.
- Add benchmarks and surveys with strong explanatory value.
- Tighten descriptions and reduce overlap.
